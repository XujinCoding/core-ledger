## 📋 API接口汇总

### 客户管理接口

| 接口 | 方法 | 路径 | 说明 |
|-----|------|------|------|
| 条件查询客户 | GET | `/api/customers` | 支持多条件筛选、分页 |
| 创建客户 | POST | `/api/customers` | 手机号唯一、地址必须村级 |
| 获取客户详情 | GET | `/api/customers/{id}` | 根据ID获取详情 |
| 修改客户 | PUT | `/api/customers/{id}` | 修改基本信息（不含地址） |
| 更新客户地址 | PUT | `/api/customers/{id}/address` | 修改地址信息 |
| 删除客户 | DELETE | `/api/customers/{id}` | 物理删除（谨慎操作） |
