## ✅ 业务规则（已确认）

### 分类规则
- [x] 分类最大层级：4层
- [x] 删除限制：有子分类或商品时不可删除

### 商品规则
- [x] 商品名称：最长200字符
- [x] 商品属性：最多5个
- [x] 商品图片：非必填，≤10M，暂存本地

### 权限控制
- [x] 只有管理员可以访问商品管理页面
- [x] 无数据权限限制
- [x] 并发编辑冲突由后端校验

### SKU管理
- [x] 由后端自动处理，前端无需管理
- [x] SKU数量限制由后端控制
- [x] 属性变更后后端自动更新SKU
