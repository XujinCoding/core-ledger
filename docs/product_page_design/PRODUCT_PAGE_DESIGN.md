# 商品页面设计文档

## 一、页面概述

商品页面是Core Ledger小程序的核心功能模块，用于管理商品分类和商品信息。采用Tab分页设计，提供简洁高效的商品管理功能。

### 设计原则
- **简洁高效**：操作流程简单，减少用户操作步骤
- **信息清晰**：层级结构明确，信息展示完整
- **易于维护**：支持快速增删改查操作
- **视觉统一**：遵循微信小程序设计规范

**3. 属性提交逻辑**
- **新增属性**: 没有id字段，后端识别为新增
- **修改属性**: 保留原有id，修改内容
- **删除属性**: 从提交数据中移除（不发送）
- **一次性提交**: 所有属性变更一次性提交到批量更新接口

**4. SKU自动处理**
- 用户只需配置属性，不用管理SKU
- 保存后后端自动计算属性组合并生成SKU
- 属性修改后，后端自动更新SKU列表
- 前端无需展示SKU预览和管理

#### 涉及的API接口
- `GET /api/products/{id}/attrs` - 获取商品属性（编辑时使用）
- `PUT /api/products/{id}/attrs/batch` - 批量更新商品属性（一次性提交所有变更）

## 四、交互流程

### 4.1 新建商品完整流程
```
1. [商品列表] Tab → 点击右上角[新增]按钮
   ↓
2. 填写商品基本信息
   ├─ 商品名称、分类
   ├─ 上传商品图片（非必填）
   ├─ 填写描述
   ├─ 规格、单位、价格
   └─ 存放位置
   ↓
3. 【可选】配置商品属性
   ├─ 添加属性（如：产地、等级等）
   └─ 为每个属性添加属性值
   ↓
4. 点击[保存]按钮
   ↓
5. 后端自动生成SKU（如有属性）
   ↓
6. 保存成功，跳转到商品详情页或列表
   ↓
完成
```

### 4.2 分类管理流程
```
创建顶级分类:
[商品分类] Tab → 点击[新增顶级分类] → 填写表单 → 保存

创建子分类:
[商品分类] Tab → 点击父分类[⋯] → 选择[添加子分类] → 填写表单 → 保存

移动分类:
[商品分类] Tab → 点击分类[⋯] → 选择[移动位置] → 选择新父分类 → 确认

删除分类:
[商品分类] Tab → 点击分类[⋯] → 选择[删除] → 确认（需检查是否有子分类和商品）
```

### 4.3 商品编辑流程
```
编辑商品信息:
[商品列表] Tab → 点击商品[⋯] → 选择[编辑商品] → 修改信息 → 保存

修改商品属性:
编辑商品页面 → 滚动到属性区域 → 添加/编辑/删除属性 → 保存
→ 后端自动更新SKU

修改分类:
[商品列表] Tab → 点击商品[⋯] → 选择[修改分类] → 选择新分类 → 确认
```

## 五、数据结构说明

### 5.1 商品分类数据
```typescript
interface CategoryVO {
  id: number;              // 分类ID
  parentId: number;        // 父分类ID（0表示顶级）
  name: string;            // 分类名称
  level: number;           // 分类层级
  sortOrder: number;       // 排序序号
  iconUrl: string;         // 分类图标URL
  memo: string;            // 备注
  status: string;          // 状态: "0"=禁用, "1"=启用
  createInstant: string;   // 创建时间
  modifyInstant: string;   // 修改时间
}

interface CategoryTreeVO extends CategoryVO {
  children: CategoryTreeVO[];  // 子分类列表
}
```

### 5.2 商品数据
```typescript
interface ProductVO {
  id: number;              // 商品ID
  categoryId: number;      // 分类ID
  categoryName: string;    // 分类名称
  name: string;            // 商品名称（最长200字符）
  imageUrl: string;        // 商品主图URL（非必填，≤10M）
  description: string;     // 商品描述
  price: number;           // 标准价格
  spec: string;            // 规格型号
  unit: string;            // 单位
  location: string;        // 存放位置
  memo: string;            // 备注
  status: string;          // 状态: "0"=禁用, "1"=启用
  attrs: ProductAttrVO[];  // 商品属性（最多5个）
  createInstant: string;   // 创建时间
  modifyInstant: string;   // 修改时间
}
```

### 5.3 商品属性数据
```typescript
interface ProductAttrVO {
  id?: number;             // 属性ID（新增时无id）
  productId: number;       // 商品ID
  attrName: string;        // 属性名称
  sortOrder: number;       // 排序序号
  values: ProductAttrValueVO[];  // 属性值列表
}

interface ProductAttrValueVO {
  id?: number;             // 属性值ID（新增时无id）
  productAttrId?: number;  // 商品属性ID
  value: string;           // 属性值
  sortOrder: number;       // 排序序号
}

// 批量更新属性请求
interface ProductAttrBatchUpdateDTO {
  attrs: ProductAttrVO[];  // 全部属性列表（新增的无id，删除的不传）
}
```
